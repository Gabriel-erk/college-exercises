Sobre a API

A API utilizada neste projeto é a Thrones API, uma API pública que fornece dados sobre o universo de Game of Thrones. Com ela, é possível acessar informações como personagens, casas, episódios e até citações da série.

A API funciona no modelo REST, ou seja, utiliza requisições HTTP simples (como GET) para retornar os dados no formato JSON. Isso permite que qualquer aplicação — seja em JavaScript, PHP, Python ou outras linguagens — consiga consumir essas informações de forma fácil.

Entre os dados disponíveis, estão informações como nome completo dos personagens, família, títulos e imagens. Esses dados podem ser utilizados para criar aplicações interativas, como listagens de personagens, páginas de detalhes ou até quizzes sobre a série.

Outro ponto importante é que a API é gratuita e não exige autenticação, o que facilita muito o uso em projetos acadêmicos e pessoais.

Neste projeto, a API foi utilizada para buscar e exibir informações dos personagens, permitindo uma navegação simples e dinâmica pelo conteúdo.
